# Personal Expense Tracker with Data Visualization

## Overview

This project is a Python-based expense tracking system that helps users:

- Track expenses
- Analyze spending
- Generate reports
- Visualize trends

---

## Features

- Expense data generation
- CSV storage
- SQLite database
- Data cleaning
- Category analysis
- Monthly analysis
- Payment method analysis
- Visualization charts
- CSV report generation
- Text report generation

---

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite
